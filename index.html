WHATSAPP CHAT ANALYZER

Export the whatsapp chat (Both induvidual chat and Group chat) to a txt file and upload here
Please click here to know how to export a chat to text file

Note:
This does not send the files to any external server for analysis ALL ANALYSIS OCCURS ON THE USER'S BROWSER
This APP maintains User Confidentiality and none of user's personal data would be provided to any server
Most of the chat analysis are based on user's chat exchanges
